# tic-tac-toe
An interactive game that allows the user to play against a computer AI (uses minmax algorithm)  the programming language used is Lua and it uses a piece of software called Love2d.
How to play this game;
1. Install love2d here: https://love2d.org/   (install win 64 bit installer)
2. Once installed create a folder on desktop called "MyGame" and drag and drop the main.lua txt file into the folder
3.  drag and drop the "MyGame" folder into the launcher on the desktop (file must have main.lua inside for it to work)
4. Enjoy!
